How to examine a CSR file

To examine a CSR file:

  1. From the Examine menu, choose Examine Certification Request.
  2. The Examine CSR dialog will appear.
  3. Select the folder where the CSR file is stored.
  4. Click on the required CSR file or type the filename into the File Name text box.
  5. Click on the Examine button.
  6. The Certification Request Details dialog will appear.
  7. After viewing the details acknowledge the dialog by pressing the OK button.

Shortcut key: Ctrl-Q.
