How to examine an SSL/TLS connection

To examine an SSL/TLS connection:

  1. From the Examine menu, choose Examine SSL/TLS Connection.
  2. The Examine SSL/TLS Connection dialog will appear.
  3. Enter the host name or IP address for the connection to be examined, for example sourceforge.net.
  4. Enter the port number for the connection to be examined, for example 443 for HTTPS.
  5. Click on the Examine button.
  6. The Certificate Details for SSL/TLS Connection dialog will appear.
  7. After viewing the details acknowledge the dialog by pressing the OK button.

Shortcut key: Ctrl-I.
