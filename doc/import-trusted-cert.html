How to import a trusted certificate

To import a trusted certificate into a keystore from file:

  1. From the Tools menu, choose Import Trusted Certificate. Alternatively click on the Import Trusted Certificate toolbar button:
  2. The Import Trusted Certificate dialog will appear.
  3. Select the folder where the certificate file is stored.
  4. Click on the required certificate file or type the filename into the File Name text box.
  5. Click on the Import button.
  6. If Portecle cannot establish a trust path from the certificate in the file to an existing self-signed trusted certificate in your keystore (or the CA certs keystore if it is enabled):
    • The Certificate Details dialog will be displayed.
    • After viewing the details acknowledge the dialog by pressing the OK button.
    • A further dialog will appear asking if you wish accept the certificate.
    • Press the Yes button if you wish to trust the certificate and import it and No if you do not. If you answer No the import will abort.
  7. The Trusted Certificate Alias dialog will appear.
  8. Enter the alias of the new trusted certificate and press OK.
  9. The new trusted certificate entry will appear in the keystore Entries table with the chosen alias.

Shortcut key: Ctrl-T.
