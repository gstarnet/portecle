How to examine a CRL file

To examine a CRL file:

  1. From the Examine menu, choose Examine CRL. Alternatively click on the Examine CRL toolbar button:
  2. The Examine CRL dialog will appear.
  3. Select the folder where the CRL file is stored.
  4. Click on the required CRL file or type the filename into the File Name text box.
  5. Click on the Examine button.
  6. The CRL Details dialog will appear.
  7. After viewing the details acknowledge the dialog by pressing the OK button.

Shortcut key: Ctrl-L.
