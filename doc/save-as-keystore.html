How to save a keystore with a different name

To save a keystore to file with a different name:

  1. From the File menu, choose Save Keystore As.
  2. The Set Keystore Password dialog is displayed.
  3. Enter the password to protect the keystore with, confirm it and press the OK button.
  4. The Save Keystore As dialog is displayed.
  5. Select the folder where the keystore file is to be saved.
  6. Type the filename into the File Name text box.
  7. Click on the Save button.
